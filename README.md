# ComfyUI-TimNodes: Google Veo 3.1 & Nano Banana (Imagen 3) for ComfyUI

Created by Tim Marinov.

## Features
-   **Veo 3.1**: Full support for Text-to-Video, Image-to-Video, First/Last Frame.
-   **Imagen 3**: High-quality image generation.
-   **Dynamic Models**: Automatically lists models available to your API key.
-   **VHS Ready**: Outputs `IMAGE` and `AUDIO` for direct video saving.

## Installation
1.  Put this folder in `ComfyUI/custom_nodes`.
2.  Add your API Key to `config.json`.
3.  Restart ComfyUI.
